## Sensor-Based Robotics Projects

A collection of embedded and electromechanical systems integrating sensors,
microcontrollers, and mechanical structures.

### Projects
- Autonomous metal-detection robot (Arduino-based)
- Color-detection system using TCS3200 sensor

### Work Performed
- Sensor integration and calibration
- Microcontroller programming
- Mechanical assembly and testing
- Functional verification and debugging

### Tools
- Arduino
- Sensors (TCS3200)
- Embedded C/C++
